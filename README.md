# image_processing_basic
Image Processing Basics (OpenCV)

Basic image and video processing examples using Python and OpenCV.

Features

This repository includes simple examples of:

Reading and displaying images

Converting images to grayscale

Video reading and playing

Image resizing and cropping

Drawing shapes (line, rectangle, circle, text)

Image merging (horizontal & vertical)

Perspective transformation

Basic computer vision experiments

Technologies
Python,OpenCV,NumPy
